## Why this site

Every line of code is a choice. This site exists because I believe those choices
should default to respecting the user — their data, their attention, their
autonomy.

No analytics. No trackers. No third-party font requests. Self-hosted everything.

## The stack

Built with [Astro](https://astro.build) for static output, React islands for
the few interactive bits, and Tailwind CSS for styling. Deployed on Vercel's
edge network.

```typescript
// Zero JavaScript by default.
// Islands only where interaction is needed.
const philosophy = 'ship less, respect more';
```
