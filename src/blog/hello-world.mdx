## Why this site

I build software with AI at the core — not as a buzzword, but as the actual
foundation. Claude Code, agents, RAG pipelines. This site is the hub where
everything connects: projects, ideas, and the lessons from shipping real
products.

No analytics. No trackers. No third-party requests. Just content.

## What I'm building

**Travelsats** — adventure tourism in Argentina powered by Bitcoin Lightning and
Nostr identity. **Xplorers** — an AI automation agency helping businesses
eliminate repetitive work. **Notarial Assistant** — a legal AI agent that edits
contracts and catches inconsistencies using RAG.

Every project starts the same way: AI from day zero, ship fast, automate
everything that can be automated.

## The stack

Built with [Astro](https://astro.build) for static output, React islands for
the few interactive bits, and Tailwind CSS for styling. Deployed on Vercel's
edge network.

```typescript
// Zero JavaScript by default.
// Islands only where interaction is needed.
const philosophy = 'AI-first, ship fast, automate the rest';
```
